Desarrollo Web con HTML y CSS

Categoría: Tecnología | Nivel: Básico

Profesor Responsable: Juan Pérez García

Duración estimada: 40 horas

Descripción General

En este curso aprenderás las bases fundamentales de la web. Comenzaremos desde cero con la estructura semántica de HTML5 y terminaremos creando sitios web estéticos y funcionales. Es el primer paso esencial para cualquier aspirante a desarrollador web.

Listado de Contenidos

  1. Introducción a Internet y la Web.
  2. Sintaxis y etiquetas básicas de HTML.
  3. Estructura semántica avanzada.
  4. Formularios y validación de datos.
  5. Introducción a CSS y selectores.
  6. Modelo de caja y Layout.

Requisitos Previos

  • Conocimientos básicos de navegación por internet.
  • No se requiere experiencia previa en programación.
  • Un editor de texto (VS Code recomendado).
